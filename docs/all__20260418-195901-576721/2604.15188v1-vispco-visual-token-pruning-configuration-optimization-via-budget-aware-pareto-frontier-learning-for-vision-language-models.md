## 摘要
视觉 Token 剪枝方法有效地缓解了视觉语言模型（VLM）在处理高分辨率图像和视频帧时产生的二次方计算增长。然而，现有方法依赖于预定义的剪枝配置，而没有确定这些配置是否达到了计算与性能的最优平衡。在这项工作中，我们引入了 VisPCO，这是一个将视觉 Token 剪枝建模为帕累托配置优化问题的新型框架，旨在自动识别最优配置。我们的方法采用连续松弛（continuous relaxation）和直通估计器（straight-through estimators）来实现基于梯度的搜索，并通过增广拉格朗日方法（Augmented Lagrangian method）进行求解。在 8 个视觉基准测试上的广泛实验表明，VisPCO 有效地逼近了通过网格搜索获得的经验帕累托前沿，并在各种剪枝方法和 VLM 架构中表现出良好的泛化性。此外，通过可学习的核函数，我们研究了逐层剪枝模式，并揭示了多步渐进式剪枝能够捕捉 VLM 的分层压缩结构，与单层方法相比，实现了更优的准确率-效率权衡。

## Abstract
Visual token pruning methods effectively mitigate the quadratic computational growth caused by processing high-resolution images and video frames in vision-language models (VLMs). However, existing approaches rely on predefined pruning configurations without determining whether they achieve computation-performance optimality. In this work, we introduce , a novel framework that formulates visual token pruning as a Pareto configuration optimization problem to automatically identify optimal configurations. Our approach employs continuous relaxation and straight-through estimators to enable gradient-based search, solved via the Augmented Lagrangian method. Extensive experiments across 8 visual benchmarks demonstrate that effectively approximates the empirical Pareto frontier obtained through grid search and generalizes well across various pruning methods and VLM architectures. Furthermore, through learnable kernel functions, we investigate layer-wise pruning patterns and reveal that multi-step progressive pruning captures VLMs' hierarchical compression structure, achieving superior accuracy-efficiency trade-offs compared to single-layer approaches.

### 2. 方法论：VisPCO 框架
VisPCO 的核心思想是将剪枝配置优化建模为一个**受约束的帕累托优化问题**，并通过梯度下降自动搜索最优解。
*   **优化目标**：在满足计算预算 $B$（以 FLOPs 衡量）的约束下，最小化剪枝后模型输出与原模型输出之间的 KL 散度（性能损失）。
*   **关键技术细节**：
    *   **增广拉格朗日法 (Augmented Lagrangian Method)**：将带约束的非凸优化问题转化为无约束目标函数，通过迭代更新拉格朗日乘子和惩罚参数来求解。
    *   **连续松弛 (Continuous Relaxation)**：针对剪枝数量不可导的问题，利用高斯核函数对 Token 重要性得分进行软插值，从而估计出可导的剪枝阈值。
    *   **直通估计器 (STE)**：在正向传播中使用离散的硬掩码（Hard Mask），在反向传播中使用 Sigmoid 软掩码的梯度，解决 Top-k 选择操作的不可导问题。
    *   **可学习核函数 (Learnable Kernels)**：引入参数化的函数（如多步 Sigmoid、指数衰减等）来建模逐层剪枝比例。这不仅缩小了搜索空间，还增强了剪枝模式的可解释性。

### 7. 优点与亮点
*   **自动化与端到端**：摆脱了繁琐的人工调参，实现了预算感知的自动化配置优化。
*   **数学严谨性**：通过增广拉格朗日法和 STE 解决了剪枝优化中长期存在的不可导难题。
*   **通用性强**：作为一个“插件式”框架，它可以提升现有任何基于重要性得分的剪枝方法的表现。
*   **高效搜索**：将原本需要数十小时的搜索过程缩短至 1 小时，极具实用价值。

### 8. 不足与局限
*   **任务覆盖**：实验主要集中在单图任务上，对于视频流或多图对话中的时空冗余优化尚待进一步验证。

*   **核函数限制**：预定义的核函数（如 Sigmoid）虽然提供了结构化先验，但也可能限制了搜索空间，无法捕捉到某些极端不规则的最优剪枝模式。
