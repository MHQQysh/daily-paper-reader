## 摘要
中长期股票分配面临重大挑战，原因在于预测结构微弱、市场状态非平稳，以及在应用交易成本、容量限制和尾部风险约束后信号会出现衰减。传统方法通常依赖单一预测器或松散耦合的“预测-分配”流水线，这限制了在分布偏移下的鲁棒性。本研究探讨了一个针对性的设计问题：在统一的滚动预测（walk-forward）循环中，将强化学习（RL）、多智能体策略种群、策略空间响应预言机（PSRO）式聚合、联赛最优反应训练、进化替换以及执行感知的检查点选择相结合，是否能提高中长期分配器的鲁棒性。所提出的框架 EvoNash-MARL 将这些组件集成在执行感知的分配循环中，并进一步引入了由方向头和风险头组成的层级策略架构、非线性信号增强、特征质量重加权以及约束感知的检查点选择。在 120 窗口滚动预测协议下，确定的 v21 配置实现了 0.7600 的平均超额夏普比率和 -0.0203 的鲁棒性得分，在内部对照组中排名第一；在 2014-01-02 至 2024-01-05 的对齐日度样本外收益中，其年化收益率为 19.6%，而 SPY 为 11.7%；在延伸至 2026-02-10 的滚动预测评估中，其收益率为 20.5%，而 SPY 为 13.5%。该框架在现实压力约束下保持了正向表现，并展现出结构化的跨市场泛化能力；然而，在 White 现实检验（WRC）和 SPA-lite 测试下，尚未建立全局显著性。因此，研究结果被视为支持更稳定的中长期训练和选择范式的证据，而非普遍优越的择时性能的证明。

传统方法通常采用单一预测器或“先预测、后分配”的松散耦合模式，在面对市场环境剧变时表现脆弱。本文的研究动机在于探索：是否可以通过将强化学习（RL）、多智能体博弈聚合（PSRO）、进化机制以及执行感知的约束选择集成到一个**闭环滚动预测（Walk-forward）循环**中，来提升分配系统在中长期维度上的稳健性。

#### 2. 方法论：核心思想与关键技术
EvoNash-MARL 框架的核心思想是将资产配置视为一个**自适应的种群博弈问题**，而非单纯的预测任务。

*   **核心架构**：
    *   **层级策略设计**：每个智能体包含一个“方向头”（预测趋势）和一个“风险头”（控制暴露），通过 $s_{k,t} = \text{sign}(\tilde{s}_{k,t})|\tilde{s}_{k,t}| \ell_{k,t}$ 实现信号生成。
    *   **PSRO 聚合层**：利用策略空间响应预言机（PSRO）计算博弈收益矩阵，通过乘法权重更新（Multiplicative Weights Update）生成元策略（Meta-strategy），以实现不同市场环境下策略的动态加权。
    *   **联赛最优反应（League Best-Response）**：引入 RL-hybrid 训练机制，让新策略针对当前种群的混合信号训练“最优反应”，增强种群的对抗性和适应力。
    *   **信号增强模块**：包含因子中性化、非线性信号放大（Power-law scaling）和特征质量重加权（基于特征在不同状态下的预测贡献度）。
*   **执行感知选择**：在验证集上不仅评估收益，还通过惩罚项（如 CVaR、最大回撤、换手率）强制筛选出符合现实交易约束的检查点。

*   **闭环集成**：将博弈论、进化算法与现实交易约束（成本、容量、Beta 暴露）深度耦合，而非事后处理。
