## 摘要
多模态大语言模型（MLLMs）由于视觉标记（token）序列中存在高度冗余，面临着巨大的计算开销。现有方法通常采用单层视觉 Transformer（ViT）特征和静态剪枝策略来解决这一问题。然而，这种固定配置在面对多样化指令时往往表现得较为脆弱。为了克服这些局限性，我们提出了 CLASP，这是一个基于类别自适应层融合和双阶段剪枝的即插即用标记缩减框架。具体而言，CLASP 首先通过多层视觉特征融合构建特定类别的视觉表示。随后，它执行双阶段剪枝，在用于保证相关性的注意力显著枢轴标记（pivot tokens）和用于保证覆盖范围的冗余感知补充标记（completion tokens）之间分配标记预算。通过类别自适应剪枝，CLASP 实现了受提示词约束的特征融合与预算分配，从而能够进行激进且鲁棒的视觉标记缩减。广泛的实验表明，CLASP 在各种基准测试、剪枝率和 MLLM 架构上均一致优于现有方法。代码将在 https://github.com/Yunkaidang/CLASP 发布。

## Abstract
Multimodal Large Language Models (MLLMs) suffer from substantial computational overhead due to the high redundancy in visual token sequences. Existing approaches typically address this issue using single-layer Vision Transformer (ViT) features and static pruning strategies. However, such fixed configurations are often brittle under diverse instructions. To overcome these limitations, we propose CLASP, a plug-and-play token reduction framework based on class-adaptive layer fusion and dual-stage pruning. Specifically, CLASP first constructs category-specific visual representations through multi-layer vision feature fusion. It then performs dual-stage pruning, allocating the token budget between attention-salient pivot tokens for relevance and redundancy-aware completion tokens for coverage. Through class-adaptive pruning, CLASP enables prompt-conditioned feature fusion and budget allocation, allowing aggressive yet robust visual token reduction. Extensive experiments demonstrate that CLASP consistently outperforms existing methods across a wide range of benchmarks, pruning ratios, and MLLM architectures. Code will be available at https://github.com/Yunkaidang/CLASP.

*   **现有局限性**：目前的标记缩减方法大多采用**静态策略**，即无论输入指令是什么，都使用固定的 ViT 层特征，并采用固定的剪枝标准（仅关注注意力相关性或仅关注特征相似度）。
*   **核心发现**：作者发现不同类别的任务对视觉抽象程度的需求不同（如计数任务需要浅层细节，语义理解需要深层特征），且对“相关性”与“覆盖范围”的平衡需求也不同。

### 2. 方法论：核心思想与关键技术
CLASP 提出了一个**类别自适应（Class-Adaptive）**的即插即用框架，包含两个核心模块：
*   **类别自适应层融合（Class-Adaptive Layer Fusion）**：
    *   利用轻量级路由器（Router）将输入文本指令分类。
    *   根据类别动态调整 ViT 不同层特征的融合权重（Softmax 归一化权重），将浅层（细节）和深层（语义）特征进行凸组合，生成最适合当前任务的视觉表示。
*   **类别自适应双阶段剪枝（Dual-Stage Pruning）**：
    *   **第一阶段（相关性保留）**：根据交叉模态注意力分数，保留 $K_1$ 个与指令最相关的“枢轴标记”（Pivots）。
    *   **第二阶段（覆盖范围补充）**：在剩余标记中，利用**冗余感知聚类（Spherical K-Means）**选择 $K_2$ 个与枢轴标记互补的标记，确保不丢失全局背景信息。
    *   **动态分配**：$K_1$ 和 $K_2$ 的比例由指令类别动态决定（如细节类任务分配更多预算给注意力枢轴）。

### 6. 主要结论与发现
*   **性能保持**：在缩减 88.9% 的视觉标记时，CLASP 仍能保留原始模型约 94.7% 的性能。
*   **效率提升**：在 POPE 基准上实现了 **1.5x 到 2.1x** 的端到端推理加速，且精度损失极小。

### 7. 优点（亮点）
*   **即插即用**：无需对模型进行重训练或微调，易于部署。
*   **多层感知**：打破了以往只用最后一层特征的惯例，通过层融合捕捉多尺度信息。
*   **双重保障**：结合了注意力机制（找重点）和聚类机制（保全面），解决了剪枝中常见的“隧道视野”问题。
*   **高度泛化**：在 LLaVA、Qwen、InternVL 等多种异构架构上均表现出色。

*   **分类依赖**：性能部分取决于意图分类器的准确性，虽然实验证明了其具有一定的容错性，但在极端未见指令下可能退化。

*   **硬件加速限制**：虽然减少了 FLOPs，但聚类操作（K-Means）在某些非优化硬件上可能无法完全转化为等比例的墙钟时间加速。
