## 摘要
我们介绍了结构化稀疏规范（S$^3$），这是一个用于定义、组合和实现结构化稀疏模式的代数框架。S$^3$ 通过三个组件来指定稀疏性：通过布局组合重塑张量的视图（View）、定义原子剪枝单元的块（Block）规范，以及稀疏决策范围（Scope）。Block 和 Scope 均支持跨张量的耦合，以实现协同稀疏化。S$^3$ 能够精确指定从细粒度 N:M 模式到粗粒度通道剪枝的多种稀疏结构，并与最优脑损伤（OBD）和最优脑外科医生（OBS）无缝集成。我们对该框架进行了数学形式化，展示了其在典型模式上的表达能力，并通过完全基于 S$^3$ 构建的结构化 OBS 和 OBD 实现进行了实验验证。实验结果表明，在常见配置的输出重构任务中，该框架优于成熟的二阶启发式方法。

## Abstract
We introduce the Structured Sparsity Specification (S$^3$), an algebraic framework for defining, composing, and implementing structured sparse patterns. S$^3$ specifies sparsity through three components: a View that reshapes the tensor via layout composition, a Block specification that defines the atomic pruning unit, and the sparsity decision Scope. Both Block and Scope support Coupling across tensors for coordinated sparsification. S$^3$ enables precise specification of diverse sparsity structures, from fine-grained N:M patterns to coarse channel pruning, and integrates seamlessly with Optimal Brain Damage (OBD) and Surgeon (OBS). We formalize the framework mathematically, demonstrate its expressiveness on canonical patterns, and validate it experimentally via structured OBS and OBD implementations built entirely on S$^3$, which surpasses well-established second order heuristics on output reconstruction across common configurations.

### 2. 论文提出的方法论
S³ 框架将结构化稀疏分解为三个核心代数组件：
*   **View（视图）**：利用布局代数（Layout Algebra）将物理张量逻辑上重塑为多维空间。它定义了坐标变换，使得复杂的硬件模式（如跨步访问）在逻辑上变得连续。
*   **Block（块）**：定义原子剪枝单元。它是被同时保留或删除的最小参数集合。
*   **Scope（范围）**：定义决策域。在每个 Scope 内，算法根据重要性得分决定保留哪些 Block（例如在 4 个 Block 中保留 2 个）。
*   **关键技术细节**：
    *   **耦合（Coupling）**：支持跨多个张量（如 Attention 的 Q/K/V/O）同步剪枝决策，确保结构一致性。
    *   **S-OBS/S-OBD 算法**：将经典的最优脑外科医生（OBS）和最优脑损伤（OBD）扩展到块级别。通过 **Schur 补（Schur Complement）** 维护逆 Hessian 矩阵，实现高效的逐行权重补偿更新。

### 7. 优点（亮点）
*   **高度通用性**：S³ 框架通过 View/Block/Scope 的组合，几乎可以描述任何已知的结构化稀疏模式，具有极强的数学优雅性。
*   **解耦设计**：将“稀疏结构定义”与“剪枝算法实现”完全解耦，开发者只需编写一次算法，即可应用于所有 S³ 定义的模式。

*   **计算开销**：S-OBS 的运行速度明显慢于 SparseGPT，对于超大规模模型（如 70B+），逐行维护逆 Hessian 的成本可能变得极高。
*   **秩亏风险**：当校准样本数小于输入维度时，逆 Hessian 的估计变得不稳定，这限制了其在小模型或极高维度层上的表现。
