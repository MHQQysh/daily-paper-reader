## 摘要
上下文学习（In-Context Learning）和思维链（Chain-of-Thought）提示提升了大语言模型（LLMs）的推理能力。这些方法通常以更长、更昂贵的提示词为代价，且其中可能包含冗余信息。基于剪枝的提示词压缩提供了一种实用的解决方案，但现有方法依赖于顺序的标记移除，计算开销较大。我们提出了 DiffuMask，这是一个基于扩散的框架，集成了分层的示例级（shot-level）和标记级（token-level）剪枝信号，通过迭代掩码预测实现快速且并行的提示词剪枝。DiffuMask 通过在每个去噪步骤中掩盖多个标记，显著加速了压缩过程。它提供了对保留内容的可调控性，在保留核心推理上下文的同时，实现了高达 80% 的提示词长度缩减。同时，它在域内、跨域和跨模型设置下均能保持或提高准确率。我们的结果表明，DiffuMask 为提示词压缩提供了一个通用且可控的框架，促进了大语言模型中更快速、更可靠的上下文推理。

## Abstract
In-Context Learning and Chain-of-Thought prompting improve reasoning in large language models (LLMs). These typically come at the cost of longer, more expensive prompts that may contain redundant information. Prompt compression based on pruning offers a practical solution, yet existing methods rely on sequential token removal which is computationally intensive. We present DiffuMask, a diffusion-based framework integrating hierarchical shot-level and token-level pruning signals, that enables rapid and parallel prompt pruning via iterative mask prediction. DiffuMask substantially accelerates the compression process via masking multiple tokens in each denoising step. It offers tunable control over retained content, preserving essential reasoning context and achieving up to 80\% prompt length reduction. Meanwhile, it maintains or improves accuracy across in-domain, out-of-domain, and cross-model settings. Our results show that DiffuMask provides a generalizable and controllable framework for prompt compression, facilitating faster and more reliable in-context reasoning in LLMs.

### 2. 方法论
*   **核心思想**：将提示词剪枝建模为一个扩散过程的逆过程。模型不直接生成文本，而是学习预测一个与原始提示词等长的二进制掩码（Binary Mask），其中 1 表示保留，0 表示删除。
*   **关键技术细节**：
    *   **分层数据生成管线**：首先进行**示例级剪枝**（Shot-level，减少示例数量），然后进行**标记级剪枝**（Token-level，使用 TA-Pruning 算法精细删除冗余词），最后通过评估过滤出能提升性能的“高质量剪枝对”作为训练数据。

    *   **损失函数设计**：采用二元交叉熵（BCE）损失，并引入**抗掩码惩罚（Anti-mask penalty）**。由于剪枝标签中“保留”远多于“删除”，该惩罚项旨在防止模型过度剪枝，确保关键推理信息不丢失。
    *   **可控推理策略**：引入 Top-k 掩码预测和置信度阈值 $\tau$，允许用户根据需求灵活调节压缩强度与准确率之间的平衡。

*   **高效压缩**：DiffuMask 能够实现高达 **80%** 的提示词长度缩减。

### 7. 优点
*   **并行加速**：利用扩散模型的并行特性解决了硬剪枝算法长期存在的计算瓶颈。
*   **分层优化**：结合了示例级和标记级的剪枝信号，比单一层级的压缩更彻底。
*   **即插即用**：生成的压缩提示词是纯文本形式，无需修改目标 LLM 的架构或权重。

### 8. 不足与局限
*   **数据获取成本高**：依赖于极其昂贵的教师模型剪枝过程来获取监督信号，限制了训练数据的规模。
*   **推理延迟**：虽然比顺序剪枝快，但 64 步扩散迭代对于“实时”应用（如毫秒级响应的对话系统）可能仍然偏慢。
*   **任务覆盖面**：目前主要验证了推理和分类任务，在长文本摘要、多轮对话或创意写作等领域的有效性尚待验证。
