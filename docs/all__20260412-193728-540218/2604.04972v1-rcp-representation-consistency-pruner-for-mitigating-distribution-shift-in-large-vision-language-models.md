## 摘要
大规模视觉语言模型（LVLMs）由于语言解码器需要处理海量的视觉标记（tokens），面临着极高的推理成本。现有的剪枝方法通常会导致显著的性能下降，因为视觉标记的不可逆移除会导致隐藏状态产生分布偏移，从而偏离预训练时的全标记模式。为了解决这一问题，我们提出了表示一致性剪枝器（Representation Consistency Pruner，简称 RCP），这是一个将累积视觉标记剪枝与延迟修复机制相结合的新型框架。具体而言，我们引入了一个交叉注意力剪枝器，利用大语言模型（LLM）的内在注意力作为基准来预测累积掩码，从而确保跨层标记减少的一致性和单调性。为了补偿由此产生的信息损失，我们设计了一个名为 DRA 的延迟修复适配器，它负责缓存被剪枝标记的精华，并专门对答案生成标记应用基于 FiLM 的调制。我们采用修复损失来使剪枝后的表示与全标记教师模型的一阶和二阶统计量相匹配。RCP 非常高效，因为它仅训练轻量级的插件模块，同时允许在推理时进行物理标记丢弃。在 LVLM 基准测试上的广泛实验表明，RCP 在仅有轻微平均准确率下降的情况下，可移除高达 88.9% 的视觉标记并减少高达 85.7% 的 FLOPs，且在多个常用基准测试上优于先前无需微调原始模型的方法。

## Abstract
Large Vision-Language Models (LVLMs) suffer from prohibitive inference costs due to the massive number of visual tokens processed by the language decoder. Existing pruning methods often lead to significant performance degradation because the irreversible removal of visual tokens causes a distribution shift in the hidden states that deviates from the pre-trained full-token regime. To address this, we propose Representation Consistency Pruner, which we refer to as RCP, as a novel framework that integrates cumulative visual token pruning with a delayed repair mechanism. Specifically, we introduce a cross-attention pruner that leverages the intrinsic attention of the LLM as a baseline to predict cumulative masks, ensuring consistent and monotonic token reduction across layers. To compensate for the resulting information loss, we design a delayed repair adapter denoted as DRA, which caches the essence of pruned tokens and applies FiLM-based modulation specifically to the answer generation tokens. We employ a repair loss to match the first and second-order statistics of the pruned representations with a full-token teacher. RCP is highly efficient because it trains only lightweight plug-in modules while allowing for physical token discarding at inference. Extensive experiments on LVLM benchmarks demonstrate that RCP removes up to 88.9\% of visual tokens and reduces FLOPs by up to 85.7\% with only a marginal average accuracy drop, and outperforms prior methods that avoid fine-tuning the original model on several widely used benchmarks.

### 1. 核心问题与整体含义（研究动机和背景）
大型视觉语言模型（LVLMs）在处理高分辨率图像时会产生海量的视觉 Token，导致推理时的计算开销（FLOPs）和内存占用（KV Cache）巨大。现有的 Token 剪枝方法虽然能减少计算量，但往往会导致显著的性能下降。
**核心痛点**：作者发现，不可逆的 Token 移除会导致模型内部隐藏状态的**分布偏移（Distribution Shift）**。这种偏移会随着模型层数的加深而累积放大，使得剪枝后的模型表现偏离了预训练时的全 Token 状态，最终损害生成质量。

### 2. 方法论：核心思想与关键技术
论文提出了 **RCP（Representation Consistency Pruner）** 框架，旨在通过“累积剪枝”与“延迟修复”维持表示的一致性。
*   **残差交叉注意力剪枝器（Residual Cross-Attention Pruner）**：
    *   结合了 LLM 固有的注意力权重、可学习的查询（Queries）以及视觉 Token 的 MLP 分数。
    *   采用**累积掩码（Cumulative Mask）**机制，确保在跨层剪枝过程中，被丢弃的 Token 不会重新出现，实现单调递减。
*   **延迟修复适配器（Delayed Repair Adapter, DRA）**：
    *   **信息缓存**：在剪枝层提取被丢弃 Token 的语义精华和位置信息。
    *   **FiLM 调制**：在模型深层（靠近输出端）插入轻量级模块，利用缓存信息通过特征线性调制（FiLM）对“答案生成 Token”的隐藏状态进行修正。
*   **分布对齐目标（Repair Loss）**：
    *   引入基于 **2-Wasserstein 距离（W2）** 的修复损失函数，通过匹配剪枝模型（学生）与全 Token 模型（教师）的一阶（均值）和二阶（方差）统计量，强制分布对齐。

*   **基座模型**：以 LLaVA-1.5-7B 为主要实验对象，并扩展至 LLaVA-1.5-13B 以验证可扩展性。

*   **评估指标**：任务准确率、Token 保留率、FLOPs 减少量、CUDA 耗时及存储占用。

*   **客观性**：实验对比了多种无需微调主干的方法，并在相同硬件环境下测试了实际推理加速比，结果较为客观、透明。

### 6. 主要结论与发现
*   **高效剪枝**：RCP 最高可移除 **88.9%** 的视觉 Token，减少 **85.7%** 的 FLOPs，而平均精度下降极小。
*   **分布修复的必要性**：实验证明，仅靠剪枝而不进行分布修复会导致深层表示严重偏离；引入二阶统计量匹配比仅匹配均值效果更好。
*   **位置敏感性**：在解码器早期进行剪枝、在靠近输出的深层进行修复是最高效的配置。
*   **性能领先**：在不微调原始模型参数的前提下，RCP 在多个指标上刷新了 SOTA，甚至在某些任务上接近全 Token 模型的表现。

### 7. 优点（亮点）
*   **非侵入性**：无需对预训练的大模型进行昂贵的全局微调，仅需训练极少量的插件模块，易于部署。
*   **理论支撑**：从分布偏移的角度切入，利用统计学中的矩匹配（Moment Matching）解决剪枝带来的负面影响。
*   **端到端加速**：不同于某些仅减少计算量但不减少内存的方法，RCP 实现了物理上的 Token 丢弃，显著降低了 KV Cache 压力。

### 8. 不足与局限
*   **任务依赖性**：虽然在通用基准上表现良好，但在极度依赖细微视觉特征的任务（如超小物体检测或复杂 OCR）中，高倍率剪枝仍可能存在风险。
*   **训练数据需求**：虽然训练开销小，但仍需要少量的多模态数据进行插件对齐，无法做到完全的“零样本（Zero-shot）”即插即用。
*   **架构局限**：目前主要在类 LLaVA 的架构上验证，对于其他非 Transformer 架构或原生多模态模型（如原生支持交错文本图像的模型）的适用性有待进一步探索。
