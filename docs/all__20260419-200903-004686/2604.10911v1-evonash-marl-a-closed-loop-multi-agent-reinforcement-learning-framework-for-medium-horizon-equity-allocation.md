## 摘要
由于预测结构微弱、非平稳的市场状态以及在应用交易成本、容量限制和尾部风险约束后信号的衰减，中长期股票配置面临着重大挑战。传统方法通常依赖于单一预测器或松散耦合的“预测-分配”流水线，这限制了其在分布偏移下的鲁棒性。本研究探讨了一个针对性的设计问题：在统一的滚动预测（walk-forward）循环中，耦合强化学习（RL）、多智能体策略种群、策略空间响应预言机（PSRO）式聚合、联赛最优反应训练、进化替换以及执行感知的检查点选择，是否能提高分配器在中长期的鲁棒性。所提出的框架 EvoNash-MARL 将这些组件集成在执行感知的分配循环中，并进一步引入了由方向头和风险头组成的分层策略架构、非线性信号增强、特征质量重加权以及约束感知的检查点选择。在 120 窗口滚动预测协议下，确定的 v21 配置实现了 0.7600 的平均超额夏普比率和 -0.0203 的鲁棒得分，在内部对照组中排名第一；在 2014-01-02 至 2024-01-05 的对齐日度样本外收益中，该框架实现了 19.6% 的年化收益率，而 SPY 为 11.7%；在截至 2026-02-10 的扩展滚动预测评估中，其收益率为 20.5%，而 SPY 为 13.5%。该框架在现实压力约束下保持了正向表现，并展示了结构化的跨市场泛化能力；然而，在 White 现实检验（WRC）和 SPA-lite 测试下，尚未建立全局显著性。因此，研究结果被视为支持更稳定的中长期训练和选择范式的证据，而非普遍优越的择时性能的证明。

*   **核心思想**：将资产配置视为一个自适应的策略种群管理问题，通过博弈论机制维持策略多样性，并利用执行感知的约束来筛选真正可部署的检查点。
