## 摘要
中长期股票配置面临重大挑战，包括预测结构微弱、非平稳市场状态，以及在应用交易成本、容量限制和尾部风险约束后信号的衰减。传统方法通常依赖单一预测器或松耦合的“预测-配置”流水线，在分布偏移下鲁棒性有限。本研究探讨了一个针对性的设计问题：在统一的滚动预测（walk-forward）循环中，将强化学习（RL）、多智能体策略种群、策略空间响应预言机（PSRO）式聚合、联赛最优响应训练、进化替换以及执行感知的检查点选择相结合，是否能提高中长期配置器的鲁棒性。所提出的框架 EvoNash-MARL 将这些组件整合到一个执行感知的配置循环中，并进一步引入了包含方向头和风险头的分层策略架构、非线性信号增强、特征质量重加权以及约束感知的检查点选择。在 120 窗口的滚动预测协议下，解析后的 v21 配置实现了 0.7600 的平均超额夏普比率和 -0.0203 的稳健得分，在内部对照组中排名第一；在 2014-01-02 至 2024-01-05 的对齐日度样本外收益中，其年化收益率为 19.6%，而标普 500 指数（SPY）为 11.7%；在延伸至 2026-02-10 的滚动预测评估中，其收益率为 20.5%，而 SPY 为 13.5%。该框架在现实压力约束下保持了正向表现，并展现出结构化的跨市场泛化能力；然而，尚未建立基于 White 现实检验（WRC）和 SPA-lite 测试的全局强显著性。因此，研究结果被视为支持一种更稳定的中长期训练与选择范式的证据，而非普遍优越的市场择时性能的证明。
