## 摘要
视觉语言模型（VLM）通过联合处理视觉和文本信息，彻底改变了多模态学习。然而，由于处理长序列视觉 Token 对计算和内存的高需求，它们面临着重大挑战。许多现有方法依赖于局部启发式准则，例如注意力分数或 Token 范数。然而，这些准则存在位置偏差和信息分散的问题，限制了它们在高剪枝率下保留核心内容的能力，并导致在视觉细节丰富的图像上性能下降。为了解决这些问题，我们提出了 SVD-Prune，一种基于奇异值分解（SVD）的无需训练、即插即用的 Token 剪枝方法。该方法通过分解视觉 Token 特征矩阵，并利用统计杠杆得分（statistical leverage scores）筛选前 K 个 Token，从而确保仅保留对主导全局方差贡献最大的 Token。实验表明，在极端的视觉 Token 预算下，SVD-Prune 始终优于先前的剪枝方法，即使在仅保留 32 和 16 个视觉 Token 的情况下也能保持强劲的性能。

## Abstract
Vision-Language Models (VLM) have revolutionized multimodal learning by jointly processing visual and textual information. Yet, they face significant challenges due to the high computational and memory demands of processing long sequences of vision tokens. Many existing methods rely on local heuristics, such as attention scores or token norms. However, these criteria suffer from positional bias and information dispersion, limiting their ability to preserve essential content at high pruning ratios and leading to performance degradation on visually detailed images. To address these issues, we propose SVD-Prune, a trainingfree, plug-and-play token pruning method based on Singular Value Decomposition. It decomposes the vision token feature matrix and selects the top-K tokens using statistical leverage scores, ensuring only tokens contributing most to the dominant global variance are preserved. Experiments show that SVD-Prune consistently outperforms prior pruning methods under extreme vision token budgets, maintaining strong performance even with 32 and 16 vision tokens.

视觉语言模型（VLM）在处理图像时通常会生成大量的视觉 Token（如 LLaVA-1.5 产生 576 个），这导致了巨大的计算开销和内存占用，成为边缘设备部署的主要瓶颈。
*   **现有方法的缺陷：** 目前主流的剪枝方法依赖注意力分数、Token 范数或跨模态相似度等局部启发式指标。
*   **发现的问题：** 作者指出，由于 LLM 解码器中的因果掩码（Causal Masking），注意力分数存在严重的**位置偏差**（靠后的 Token 参与计算的次数少，得分天然偏低）。此外，局部指标难以捕捉全局视觉结构，导致在高剪枝率下性能大幅下降。
*   **研究目标：** 开发一种无需训练、不受位置偏差影响、能从全局视角识别关键信息的视觉 Token 剪枝方案。

### 2. 论文提出的方法论：SVD-Prune
核心思想是利用**奇异值分解（SVD）**从全局方差的角度评估 Token 的重要性，而非依赖不稳定的注意力权重。

*   **关键技术流程：**
    1.  **全局模式提取：** 对视觉编码器输出的特征矩阵 $F \in \mathbb{R}^{T \times D}$ 进行 SVD 分解，得到左奇异向量 $U$、奇异值矩阵 $\Sigma$ 和右奇异向量 $V^\top$。
    2.  **主导方差截断：** 计算累积解释方差比，选择最小的 $k$ 个成分，使得保留的方差比例达到超参数 $\epsilon$（通常在 0.7 到 0.95 之间）。这能过滤掉噪声和冗余信息。
    3.  **统计杠杆得分（Leverage Scores）：** 计算每个 Token 在前 $k$ 个主成分上的投影平方和。得分越高，代表该 Token 对全局方差结构的贡献越大，即越具有代表性。
    4.  **Token 筛选与重排：** 根据杠杆得分降序排列，选取满足累积重要性阈值的最小 Token 集合。最后将保留的 Token 按原始空间顺序重排，以保持位置编码的一致性。

### 6. 主要结论与发现
*   **全局优于局部：** 基于 SVD 的全局方差分析比基于注意力的局部启发式方法更稳健，能有效规避位置偏差。

*   **冗余性证实：** 实验再次证明 VLM 的视觉 Token 存在极高冗余，多模态推理在很大程度上是由文本驱动的，少量的关键视觉 Token 即可支撑大部分推理任务。

### 7. 优点
*   **无需训练（Plug-and-Play）：** 现有的 VLM 无需任何微调即可直接应用，部署成本极低。
*   **数学基础扎实：** 利用 SVD 和杠杆得分（Leverage Scores）提供了比直觉启发式更严谨的统计学解释。
*   **解决位置偏差：** 成功解决了 LLM 因果掩码导致的注意力评分不公问题。
*   **极端高效：** 在极低预算下依然保持了可用的精度，非常适合资源受限的嵌入式设备。

### 8. 不足与局限
*   **计算开销：** 虽然减少了 LLM 的推理开销，但 SVD 分解本身在 CPU/GPU 上存在一定的计算成本（尽管对于 576x1024 规模的矩阵来说相对较小）。
*   **超参数敏感性：** 解释方差比例 $\epsilon$ 的选择可能需要针对不同任务进行微调。
*   **任务覆盖面：** 实验主要集中在 VQA 类任务，尚未在长视频理解或更复杂的视觉定位（Grounding）任务上进行广泛验证。
*   **模型局限：** 主要在 LLaVA-1.5 上验证，对于采用不同视觉编码器（如 SigLIP）或其他架构的 VLM 表现尚待观察。
