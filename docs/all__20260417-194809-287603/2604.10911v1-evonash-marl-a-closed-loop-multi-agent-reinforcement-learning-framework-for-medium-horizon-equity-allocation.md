## 摘要
中长期股票分配面临重大挑战，包括预测结构微弱、非平稳市场状态，以及在应用交易成本、容量限制和尾部风险约束后信号的退化。传统方法通常依赖单一预测器或松耦合的“预测-分配”流水线，这限制了在分布偏移下的鲁棒性。本研究探讨了一个针对性的设计问题：在统一的滚动预测（walk-forward）循环中，将强化学习（RL）、多智能体策略种群、策略空间响应预言机（PSRO）式聚合、联赛最优反应训练、进化替换以及执行感知的检查点选择相结合，是否能提高中长期分配器的鲁棒性。所提出的框架 EvoNash-MARL 将这些组件集成在执行感知的分配循环中，并进一步引入了包含方向头和风险头的分层策略架构、非线性信号增强、特征质量重加权以及约束感知的检查点选择。在 120 窗口的滚动预测协议下，确定的 v21 配置实现了 0.7600 的平均超额夏普比率和 -0.0203 的鲁棒得分，在内部对照组中排名第一；在 2014-01-02 至 2024-01-05 的对齐日度样本外收益中，其年化收益率为 19.6%，而标普 500 指数（SPY）为 11.7%；在延伸至 2026-02-10 的滚动预测评估中，其收益率为 20.5%，而 SPY 为 13.5%。该框架在现实压力约束下保持了正向表现，并展现出结构化的跨市场泛化能力；然而，尚未建立 White 现实检验（WRC）和 SPA-lite 测试下的全局强显著性。因此，研究结果被视为支持更稳定的中长期训练与选择范式的证据，而非普遍优越的市场择时表现的证明。

EvoNash-MARL 框架的核心思想是将资产配置建模为一个多智能体博弈与进化的闭环系统：
*   **分层策略架构**：每个智能体包含一个**方向头（Direction Head）**预测趋势，以及一个**风险头（Risk Head）**动态调整杠杆和暴露。
*   **博弈聚合层（PSRO）**：引入策略空间响应预言机（PSRO）机制，通过计算收益矩阵的纳什均衡来更新元策略（Meta-strategy），从而对不同策略进行加权聚合，而非简单的平均。
*   **联赛最优反应（League Best-Response）**：通过 RL-hybrid 方法训练针对当前种群混合信号的“最优反应”候选者，并将其注入种群以保持竞争力和适应性。
*   **信号增强模块**：包含因子中性化（Factor Neutralization）、非线性信号放大、以及基于特征质量的动态重加权。
*   **执行感知选择**：在检查点（Checkpoint）选择阶段，不仅考虑原始收益，还强制引入交易成本、容量限制、Beta 暴露和尾部风险（CVaR）等约束，剔除“回测好看但无法落地”的策略。
