## 摘要
Transformer 在视觉和语言领域都是强大的基准模型，因为自注意力机制能够捕捉 token 之间的长程依赖关系。然而，自注意力的计算成本随 token 数量呈平方级增长。Patch 剪枝通过估计每个 patch 的重要性并移除冗余 patch 来降低这一成本。为了识别用于剪枝的信息丰富的 patch，我们引入了一种基于注意力分布 Shannon 熵的标准。低熵 patch 接收到的是选择性且集中的注意力，因此被视为重要的并予以保留；而高熵 patch 的注意力分散在多个位置，被视为冗余。我们还将该标准从 Shannon 熵扩展到 Rényi 熵，后者强调尖锐的注意力峰值，并支持能够适应任务需求和计算限制的剪枝策略。在 patch 选择至关重要的细粒度图像识别实验中，我们的方法在保持准确性的同时减少了计算量。此外，通过 Rényi 熵度量调整剪枝策略可以获得进一步的收益，并改善准确性与计算量之间的权衡。

## Abstract
Transformers are strong baselines in both vision and language because self-attention captures long-range dependencies across tokens. However, the cost of self-attention grows quadratically with the number of tokens. Patch pruning mitigates this cost by estimating per-patch importance and removing redundant patches. To identify informative patches for pruning, we introduce a criterion based on the Shannon entropy of the attention distribution. Low-entropy patches, which receive selective and concentrated attention, are kept as important, while high-entropy patches with attention spread across many locations are treated as redundant. We also extend the criterion from Shannon to Rényi entropy, which emphasizes sharp attention peaks and supports pruning strategies that adapt to task needs and computational limits. In experiments on fine-grained image recognition, where patch selection is critical, our method reduced computation while preserving accuracy. Moreover, adjusting the pruning policy through the Rényi entropy measure yields further gains and improves the trade-off between accuracy and computation.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：Vision Transformer (ViT) 的自注意力机制计算复杂度随 Patch（补丁）数量呈平方级增长，导致在处理高分辨率图像或细粒度识别任务时计算开销巨大。
*   **研究动机**：现有的 Patch 剪枝方法主要依赖注意力权重的幅度（Magnitude）来衡量重要性。然而，高权重并不一定代表高信息量（可能注意力非常分散），且权重大小无法直接量化模型对特定区域的确定性。
*   **核心思路**：作者提出利用**信息论中的“熵”**来衡量 Patch 的重要性。直观上，如果一个 Patch 的注意力分布非常集中（低熵），说明它关注的是关键的前景目标；如果分布弥散（高熵），则通常对应背景或冗余信息。

### 2. 论文提出的方法论
*   **核心思想**：将注意力分布视为概率分布，通过计算每个 Patch 的注意力熵（Attention Entropy）来识别并保留信息丰富的 Patch，剔除冗余 Patch。
*   **关键技术细节**：
    1.  **注意力分布定义**：对于每个 Patch $x_i$，计算其对所有其他 Patch 的注意力得分，通过 Softmax 归一化得到分布 $a(x_i)$。
    2.  **Shannon 熵准则**：计算 $H(x_i) = -\sum a_j \log a_j$。低熵表示注意力集中，高熵表示注意力分散。
    3.  **Rényi 熵扩展**：引入广义熵 $H_\alpha(x_i) = \frac{1}{1-\alpha} \log \sum a_j^\alpha$。通过调节阶数参数 $\alpha$，可以控制对注意力分布中“峰值”的敏感度。当 $\alpha > 1$ 时，更强调高概率事件（即更关注最尖锐的注意力中心）。
*   **算法流程**：
    *   在预训练 ViT 的微调阶段，于特定层（如第 4, 7, 10 层）插入剪枝模块。
    *   计算当前层所有 Patch 的注意力熵。
    *   按熵值从小到大排序，保留前 $100r\%$（$r$ 为保留率）的 Patch。
    *   仅将保留的 Patch 和 Class Token 传向下一层。
