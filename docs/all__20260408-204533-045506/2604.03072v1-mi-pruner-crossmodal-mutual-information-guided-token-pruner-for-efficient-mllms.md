## 摘要
对于多模态大语言模型（MLLMs），视觉信息相对于文本而言较为稀疏。因此，为了实现高效推理，视觉剪枝研究应运而生。目前的方法通常基于视觉编码器或大语言模型（LLM）解码器中的注意力分数来衡量 Token 的重要性，然后选择注意力分数较高的视觉 Token 并剪掉其他 Token。在本文中，我们采用了一种不同且更精准的方法。我们不依赖于特定机制的信号，而是在视觉和文本特征交互之前，直接计算它们之间的互信息（MI）。这使我们能够在特征层面显式地衡量跨模态依赖关系。我们的 MI-Pruner 简单、高效且具有非侵入性，不需要访问内部注意力图或进行架构修改。实验结果表明，我们的方法在极低延迟的情况下优于以往基于注意力的剪枝方法。

## Abstract
For multimodal large language models (MLLMs), visual information is relatively sparse compared with text. As a result, research on visual pruning emerges for efficient inference. Current approaches typically measure token importance based on the attention scores in the visual encoder or in the LLM decoder, then select visual tokens with high attention scores while pruning others. In this paper, we pursue a different and more surgical approach. Instead of relying on mechanism-specific signals, we directly compute Mutual Information (MI) between visual and textual features themselves, prior to their interaction. This allows us to explicitly measure crossmodal dependency at the feature levels. Our MI-Pruner is simple, efficient and non-intrusive, requiring no access to internal attention maps or architectural modifications. Experimental results demonstrate that our approach outperforms previous attention-based pruning methods with minimal latency.

#### 1. 核心问题与整体含义（研究动机和背景）
多模态大语言模型（MLLM）在处理高分辨率图像时，视觉 Token 的数量远超文本 Token，且视觉信息本身具有高度冗余性，导致推理时的计算开销（尤其是自注意力的二次方复杂度）巨大。
*   **现有挑战**：目前的剪枝方法多依赖视觉编码器或 LLM 解码器内部的**注意力分数**。然而，注意力分数存在位置偏差、注意力汇聚（Attention Sinks）以及多头多样性等噪声，且提取注意力矩阵会带来额外的内存和延迟开销，与 FlashAttention 等加速技术不兼容。
*   **核心动机**：回归信息论本质，不依赖模型内部机制信号，直接在特征投影空间通过计算视觉与文本特征间的**互信息（Mutual Information, MI）**来衡量 Token 的重要性，实现一种简单、高效且非侵入性的剪枝方案。

#### 2. 方法论：核心思想与技术细节
MI-Pruner 将 Token 剪枝建模为一个受基数约束的单调次模函数最大化问题。
*   **核心思想**：在视觉特征经过投影层（Projector）后、进入 LLM 解码器前，计算视觉 Token ($v_i$) 与文本 Token ($t_j$) 之间的互信息。
*   **关键步骤**：
    1.  **相似度计算**：将视觉和文本嵌入归一化到单位超球面上，计算视觉-文本（跨模态）和视觉-视觉（模态内）的相似度矩阵。
    2.  **概率分布推导**：利用 Softmax 函数将相似度转化为条件概率分布 $p(t_j|v_i)$，并基于全概率公式推导边缘分布。
    3.  **点向互信息（PMI）计算**：计算每个视觉 Token 相对于文本查询的语义对齐度（跨模态 PMI）以及相对于已选视觉 Token 的冗余度（模态内 PMI）。
    4.  **贪心搜索选择**：定义评分函数 $S_i = \lambda S_{cross} - (1-\lambda) S_{self}$，通过贪心策略依次选择边际增益最大的视觉 Token，直到达到预设的保留预算（$N_{keep}$）。
*   **算法特性**：该方法是模型无关的（Model-agnostic），不需要修改模型架构，也不需要访问注意力图。

#### 7. 优点（亮点）
*   **理论完备**：基于信息论中的互信息和次模函数最大化，相比启发式的注意力剪枝更具解释性。

*   **Prompt 感知**：能够根据不同的查询动态调整剪枝策略，实现“按需剪枝”。
