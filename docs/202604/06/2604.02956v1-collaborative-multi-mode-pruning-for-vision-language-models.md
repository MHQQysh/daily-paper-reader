## 摘要
视觉-语言模型（VLMs）在统一的 Transformer 架构下取得了飞速发展，但由于计算复杂度高，在资源受限设备上的部署仍然面临挑战。虽然剪枝已成为压缩 VLMs 的有效技术，但现有方法主要集中在单一模式上，即仅对参数或 Token 进行剪枝，忽略了对各模式中固有冗余的充分探索，这导致在高剪枝率下性能大幅下降。为了解决上述局限性，我们提出了协同多模式剪枝（CoMP），这是一种专为 VLMs 设计的新型框架，通过执行联合参数和 Token 剪枝来实现。具体而言，我们首先设计了一种协同重要性指标（CIM），用于研究耦合参数与 Token 之间的相互干扰。它将 Token 的不同显著性纳入参数重要性得分的计算中，同时减轻了被剪枝参数对 Token 重要性得分的影响。此外，我们开发了一种多模式剪枝策略（MPS），将整个剪枝过程分解为一系列剪枝阶段，在每个阶段中，我们根据剪枝成本估计不同剪枝模式的优先级，并自适应地切换到最优模式。此外，MPS 整合了历史成本和随机探索，以实现稳定的剪枝过程并避免局部最优。在各种视觉-语言任务和模型上的广泛实验表明，与最先进的方法相比，我们的方法在高剪枝率下有效地提升了性能。源代码可在 https://github.com/Wuzimeng/CoMP.git 获取。

## Abstract
Vision-Language Models (VLMs) have advanced rapidly within the unified Transformer architecture, yet their deployment on resource-constrained devices remains challenging due to high computational complexity. While pruning has emerged as an effective technique for compressing VLMs, existing approaches predominantly focus on a single mode by pruning either parameters or tokens, neglecting fully exploring the inherent redundancy in each mode, which leads to substantial performance degradation at high pruning ratios. To address the above limitations, we propose Collaborative Multi-Mode Pruning (CoMP), a novel framework tailored for VLMs by performing joint parameter and token pruning. Specifically, we first design a Collaborative Importance Metric (CIM) that investigates the mutual interference between the coupled parameters and tokens. It incorporates distinct significance of tokens into the computation of parameter importance scores, while simultaneously mitigating the affect of pruned parameters on token importance scores. Moreover, we develop a Multi-Mode Pruning Strategy (MPS) that decomposes the overall pruning process into a sequence of pruning stages, while in each stage we estimate the priory of different pruning modes based on their pruning cost and adaptively shift to the optimal one. Additionally, MPS integrates the historical cost and random exploration, in order to achieve a stable pruning process and avoid local optimum. Extensive experiments across various vision-language tasks and models demonstrate that our method effectively promotes the performance under high pruning ratios by comparing to the state-of-the-art approaches. The source code is available at https://github.com/Wuzimeng/CoMP.git.

#### 1. 核心问题与整体含义（研究动机和背景）
随着视觉-语言模型（VLMs，如 BLIP, CLIP, LLaVA）规模的增大，其在资源受限设备上的部署面临巨大的计算开销。现有的模型压缩技术主要分为**参数剪枝**（减少特征维度）和**Token剪枝**（减少序列长度）。
*   **研究动机**：作者发现单一维度的剪枝无法充分挖掘模型内部的冗余。简单的将两者结合（联合剪枝）效果不佳，主要存在两个挑战：
    1.  **重要性度量的互干扰**：参数的重要性计算受冗余 Token 影响，而 Token 的重要性评估又依赖于即将被剪掉的参数。
    2.  **剪枝模式应用僵化**：在不同的剪枝阶段，参数和 Token 的冗余程度是动态变化的，固定比例或顺序的剪枝会导致次优解。

#### 2. 方法论：核心思想与关键技术
论文提出了 **Collaborative Multi-Mode Pruning (CoMP)** 框架，核心包含两个关键组件：
*   **协同重要性指标 (CIM)**：
    *   **Token 加权的参数重要性**：在计算参数重要性得分时，引入 Token 的显著性权重，降低不重要 Token 对参数评估的干扰。
    *   **自校正的 Token 重要性**：将参数剪枝掩码直接应用于注意力权重矩阵，消除冗余参数（如无用的注意力头）对 Token 重要性排名的误导。
*   **多模式剪枝策略 (MPS)**：
    *   **成本感知模式切换**：定义了 5 种模式（视觉/语言/跨模态参数、视觉/语言 Token），根据“验证集精度变化 / FLOPs 减少量”计算剪枝成本，优先选择成本最低（即性能损失最小）的模式。
    *   **探索与历史信息**：引入随机探索机制（避免陷入局部最优）和历史执行时间戳（通过指数移动平均 EMA 稳定剪枝过程），确保剪枝过程的鲁棒性。

*   **消融实验**：详细验证了 CIM（参数加权、Token 校正）和 MPS（成本感知、随机探索、历史信息）各子模块的贡献。

    *   实验设计对比了多种 SOTA 方法，并提供了推理延迟的实测数据，具有较高的客观性和说服力。

#### 6. 主要结论与发现
*   **高剪枝率优势**：CoMP 在高剪枝率（如 0.8 或 0.85）下表现尤为突出。例如在 NLVR2 任务上，0.85 剪枝率下比 MADTP 提升了 3.51% 的准确率。
*   **协同效应**：通过 CIM 解决参数与 Token 间的干扰，能显著提升联合剪枝的基准线。
*   **动态冗余**：模型冗余并非静态，MPS 证明了在剪枝不同阶段自适应切换模式比固定策略更有效。

#### 7. 优点
*   **视角全面**：首次深入探讨了参数剪枝与 Token 剪枝在 VLM 中的相互干扰问题。
*   **自适应强**：MPS 策略不需要人工预设复杂的剪枝比例，能够根据模型状态自动寻找最优压缩路径。
*   **通用性好**：不仅适用于中等规模模型，也能无缝扩展到 LLaVA 等大规模多模态模型。

#### 8. 不足与局限
*   **计算开销**：MPS 策略在每个剪枝阶段需要评估验证集精度，虽然作者通过子采样验证集降低了开销，但相比于训练后剪枝（PT-Pruning），其搜索过程仍需一定的计算资源。
*   **应用限制**：目前主要针对 Transformer 架构，对于非 Transformer 结构的视觉编码器（如 CNN）或新型架构的适用性尚未讨论。
*   **超参数依赖**：虽然 MPS 减少了比例设置，但引入了如探索率 $\rho$、衰减因子 $\lambda$ 等新的超参数，尽管实验证明其敏感度较低，但在极端新场景下可能需要微调。
