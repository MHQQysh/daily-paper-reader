## 摘要
尽管大型视觉语言模型 (LVLMs) 展示了卓越的多模态能力，但处理高分辨率视觉 token 的二次方计算成本仍然是一个关键瓶颈。虽然近期的 token 减少策略试图加速推理，但这些方法未能充分利用注意力值，且无法解决 token 冗余问题。更关键的是，它们忽视了 LVLMs 中固有的“注意力偏移”现象，该现象会导致 token 注意力分数产生偏差。在这项工作中，我们提出了 ASAP，这是一种新型的、无需训练且兼容 KV-Cache 的剪枝方案，全面解决了上述局限性。首先，我们通过利用动态双向软注意力掩码来缓解注意力偏移，确保选择真正具有信息量的 token，而非简单的基于注意力的选择。其次，我们认为 token 集合中的高语义冗余会降低性能。因此，我们引入了一个加权软合并组件来合并语义相似的 token，仅为后续层保留特征最密集的视觉补丁。ASAP 实现了视觉上下文的几乎无损压缩，在保留 LLaVA-NeXT-7B 原始性能 99.02% 的同时，大幅削减了约 80% 的计算 FLOPs。

## Abstract
While Large Vision-Language Models (LVLMs) demonstrate exceptional multi-modal capabilities, the quadratic computational cost of processing high-resolution visual tokens remains a critical bottleneck. Though recent token reduction strategies attempt to accelerate inference, such methods inadequately exploit attention values and fail to address token redundancy. More critically, they overlook the ``attention shift'' phenomenon inherent in LVLMs, which skews token attention scores. In this work, we propose ASAP, a novel training-free, KV-Cache-compatible pruning recipe that comprehensively addresses these limitations. First, we mitigate the attention shift by utilizing a dynamic bidirectional soft attention mask, ensuring the selection of genuinely informative tokens rather than naive attention-based selection. Second, we posit that high semantic redundancy within the token set degrades performance. We therefore introduce a weighted soft merging component that merges semantically similar tokens, preserving only the most feature-dense visual patches for subsequent layers. ASAP achieves virtually lossless compression of visual context, retaining 99.02% of the original LLaVA-NeXT-7B performance while aggressively slashing computational FLOPs by ~80%.

### 1. 核心问题与研究动机
*   **研究背景**：大型视觉语言模型（LVLM）在处理高分辨率图像时，视觉标记（Token）数量剧增（如 LLaVA-NeXT 达 2304 个），导致 Transformer 的二次方计算开销成为推理瓶颈。
*   **核心问题**：现有的 Token 剪枝方法存在两大缺陷：
    1.  **注意力偏移（Attention Shift）**：由于旋转位置嵌入（RoPE）的距离衰减特性，靠后的视觉 Token 往往获得虚高的注意力分数，而靠前的 Token 被低估，导致基于原始注意力的剪枝会产生空间偏置（如只关注图像底部）。
    2.  **语义冗余**：图像中存在大量低信息密度的区域（如天空、背景），简单的剪枝无法有效处理这些冗余。
*   **研究动机**：开发一种无需训练、兼容 KV 缓存且能感知注意力偏移的高效剪枝方案。

### 2. 论文提出的方法论（ASAP）
ASAP 包含三个核心组件，旨在实现“注意力偏移感知”的视觉上下文压缩：
*   **显著性引导的双向注意力（SG-BiMask）**：
    *   **核心思想**：打破标准的因果掩码（Causal Mask），允许视觉 Token 之间进行双向信息流动。
    *   **技术细节**：利用预 Softmax 的对齐矩阵计算 Token 的原始显著性，通过 Min-Max 归一化投影到 [0, 1] 区间。引入一个动态的软双向掩码 $M_{bidir}$，根据显著性分数对“未来”Token 的可见度进行加权惩罚，从而抵消 RoPE 带来的位置偏置。
*   **显著性加权合并（Salience-Weighted Consolidation）**：
    *   **核心思想**：在保留关键 Token 的同时，合并语义相似的冗余 Token。
    *   **技术细节**：计算 Token 间的余弦相似度，将相似 Token 合并到显著性更高的“锚点”Token 中。合并过程采用基于显著性分数的凸组合（Convex Combination），确保特征不会被稀释。
*   **预算重新分配（Budget Reallocation）**：
    *   **核心思想**：优化固定 Token 预算的利用率。
    *   **技术细节**：合并操作会腾出预算位置，ASAP 会从之前被剪掉的 Token 池中捞回显著性最高的 Token 填补空位，确保最终序列的信息密度最大化。
