## 摘要
注意力汇聚（Attention sinks）和巨量激活（massive activations）是 Transformer 模型中反复出现且密切相关的现象。现有研究主要集中在前向传播过程，因此尚不清楚两者之间的联系是直接的，还是由某种训练时机制介导的。我们从反向传播的角度研究了这一问题。通过实证和理论分析，我们证明在因果掩码（causal mask）下，注意力汇聚会诱导显著的梯度集中，我们将其称之为梯度汇聚（gradient sinks）。此外，在采用 RMSNorm 的 Pre-norm 架构中，巨量激活可以被理解为训练过程中对这种局部梯度压力的一种自适应响应。为了验证这一假设，我们引入了 V-scale，这是一种调整值路径（value-path）反向传播梯度的修改方法。在预训练的 V-scale 模型中，注意力汇聚得以保留，而巨量激活则被抑制。这些结果支持了这样一种解释：梯度汇聚是连接注意力汇聚和巨量激活的关键训练时中介。

## Abstract
Attention sinks and massive activations are recurring and closely related phenomena in Transformer models. Existing studies have largely focused on the forward pass, making it unclear whether their connection is direct or mediated by a training-time mechanism. We study this question from the perspective of backpropagation. Empirically and theoretically, we show that under causal mask, attention sinks can induce pronounced gradient concentration, which we term gradient sinks. Furthermore, in pre-norm architectures with RMSNorm, massive activations can be understood as an adaptive response to this localized gradient pressure during training. To test this hypothesis, we introduce V-scale, a modification that adjusts value-path backpropagated gradients. In pretrained V-scale models, attention sinks are preserved whereas massive activations are suppressed. These results support the interpretation that gradient sink is a key training-time mediator linking attention sinks and massive activations.

*   **核心问题**：在 Transformer（尤其是 Pre-norm 架构）中，为什么优化过程会学习到极大的激活值（MA）？
*   **背景**：
    *   **AS**：模型倾向于将大量注意力权重集中在少数 Token（通常是第一个 Token）上。
    *   **MA**：在相同位置，激活值的范数会变得异常巨大。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：提出“**梯度汇聚（Gradient Sinks, GS）**”概念。在因果掩码下，后续所有 Token 的梯度都会汇集到初始 Token 的 Key 和 Value 路径上。MA 的出现并非偶然，而是模型为了抵消这种局部梯度压力、稳定训练而产生的一种自适应响应。
*   **关键技术细节**：
    *   **梯度汇聚理论**：证明了在因果掩码下，Value 路径的梯度是后续所有梯度以注意力权重为系数的加权和。AS 越强，该位置积累的梯度（GS）就越显著。
    *   **RMSNorm 的压缩作用**：理论证明 RMSNorm 的雅可比矩阵范数与输入激活值的范数成反比。这意味着，**激活值越大，反向传播回来的梯度就被压缩得越厉害**。
    *   **V-scale 机制（干预手段）**：为了验证因果关系，作者设计了 V-scale 模块：$\hat{v}_j = \phi(\|v_j\|_2^2) v_j$，其中 $\phi(r) = r/(r+C)$。
        *   该机制在 Value 路径上充当“梯度阀门”。
        *   对于范数较小的 Token（如 Sink Token 的 Value 通常较小），它会显著衰减其反向传播的梯度，从而减轻对 MA 的需求。
